#The Goal of This Lab
The overall goal of this lab is to create a permanent Active Directory environment on my Proxmox nodes. I want to use this AD network for:

  Penetration Testing Practice - Simulating real-world attacks and defenses

  Splunk Log Analysis - Generating security logs for practicing investigations in Splunk

  Hands-on Learning - Gaining experience in:

    Setting up and configuring an Active Directory network

    Managing AD infrastructure

    Understanding AD security and administration

This Lab Objectives
  Build a realistic Active Directory domain on a different network

  Configure domain controllers, member servers, and client workstations

  Implement common enterprise services (DNS, DHCP, File Sharing, etc.)
